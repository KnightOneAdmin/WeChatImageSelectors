# WeChatImageSelectors
仿微信图片选择器

![](http://odox9bbkw.bkt.clouddn.com/2016-09-11%2017_22_37.gif)

## 项目介绍
* 1.google  list(Lists.newArrayList)
* 采用 picasso 加载图片
* 采用butterknife 注解
* 采用 RecyclerView 实现首页新闻多 item 布局
* 自定义控件 DragGridView 
* 自定义控件 SquareLayout 

## 关于
本demo 主是要项目中做bbs功能发贴。 选择图片的时候用到图片选择器，所以把项目中的抽出来，希望有用到的童鞋可以帮助到。
